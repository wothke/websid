This is the latest version of Tiny'R'Sid.

2015 by Juergen Wothke (The source code can be found here.)

This page does not use any plugins but is based exclusively on HTML5/JavaScript/WebAudio. You'll need a reasonably recent browser to make it play the music.

The music samples are completely calculated within JavaScript by running the Tiny'R'Sid C64 emulator logic (playing original C64 program files from the HVSC). You can bring your own music files by drag&dropping them on the below C64 image. (Plays *.sid, *.psid, *.rsid and Compute! *.mus music files.)
